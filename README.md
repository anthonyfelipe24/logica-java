🧠 Desafios de Lógica de Programação & Algoritmos

> ### 🧰 Foco do Repositório:
> `Java (OpenJDK)` | `Lógica de Programação` | `Algoritmos`

Espaço criado exclusivamente para registrar a resolução de desafios lógicos, exercícios de algoritmos e testes de código puros utilizando **Java**. A proposta deste repositório é aplicar na prática os conceitos teóricos estudados na graduação de ADS e na Maratona Java do DevDojo, blindando a base de fluxo de dados.

---

## 🚀 Estrutura de Estudos Técnico

O foco atual das resoluções abrange os pilares fundamentais da computação:
* **Estruturas de Controle:** Tomadas de decisão e condicionais encadeadas (`if`, `else if`, `else`, `switch`).
* **Estruturas de Repetição:** Laços de iteração (`for`, `while`, `do-while`).
* **Estruturas de Dados Básicas:** Criação, manipulação e leitura de Arrays (Vetores e Matrizes).

---

## 🛠️ Como Executo os Desafios

Todos os códigos deste repositório utilizam Java puro (Standard Edition) e são executados diretamente via terminal para validação de entrada e saída de dados.

1. Compile o arquivo: `javac NomeDoExercicio.java`
2. Execute a aplicação: `java NomeDoExercicio`
